# 📜 CanonMaster · 典校大师
专为人文社科领域学术群体打造的智能参考文献排版转换工具，基于智谱GLM-5大语言模型，一键将格式混乱的参考文献转换为符合学术标准的规范格式。

## ✨ 核心功能
- 支持GB/T 7714-2015通用国标，细分文末/脚注双模式
- 内置《文学遗产》等专业期刊专属体例，一键适配投稿要求
- 支持自定义规则，满足个性化排版需求
- 桌面端/移动端自动适配，全端体验一致
- 本地浏览器数据存储，充分保障文献隐私安全
